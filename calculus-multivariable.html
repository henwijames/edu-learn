Multivariable Calculus

Extend your calculus knowledge to functions of multiple variables and explore vector calculus concepts.

Partial Derivatives

Functions of Multiple Variables

Learn how to differentiate functions with respect to one variable while holding others constant.

Key Concepts:

  • Partial derivatives
  • Higher-order partial derivatives
  • Chain rule for multivariable functions
  • Directional derivatives

Example:

Find ∂f/∂x and ∂f/∂y for f(x,y) = x²y + sin(xy)

Solution:

  1. ∂f/∂x = 2xy + y·cos(xy)
  2. ∂f/∂y = x² + x·cos(xy)

Multiple Integrals

Integration in Higher Dimensions

Extend integration concepts to functions of multiple variables and learn to calculate volumes and masses.

Key Concepts:

  • Double integrals
  • Triple integrals
  • Change of variables
  • Applications in physics

Example:

Evaluate ∫∫(x + y) dA over the region R: 0 ≤ x ≤ 1, 0 ≤ y ≤ 2

Solution:

  1. Set up iterated integral
  2. Integrate with respect to y first
  3. Then integrate with respect to x
  4. Result: 3

Vector Calculus

Vector Fields and Theorems

Explore vector fields, line integrals, and fundamental theorems of vector calculus.

Key Concepts:

  • Vector fields
  • Line integrals
  • Surface integrals
  • Green's Theorem
  • Stokes' Theorem

Example:

Calculate the line integral of F = (y, -x) along the unit circle

Solution:

  1. Parameterize the circle
  2. Set up the line integral
  3. Evaluate using trigonometric identities
  4. Result: -2π
